# 📊 AdventureWorks Dashboard Project

## 🚀 Project Overview

This project is a **Power BI Dashboard** for analyzing:

* Sales Performance
* Customer Insights
* Product Analysis
* Regional Sales Distribution

---

## 📌 Key Features

* 📈 Revenue & Profit Analysis
* 👥 Customer Segmentation
* 🌍 Geographic Sales Map
* 📦 Product Performance
* 📊 KPI Indicators (Revenue, Orders, Returns)
